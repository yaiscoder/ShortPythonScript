# Participant registration

## Description
This program is a partcipant registration system for a computer course.
It is a program that collects basic information

## How it works
1. The program asks for the number of participants
2. It uses a 'for' loop to request the information for each participant:
- Name
- Age
- Basic computer knowledge
3. The program checks if the participant is older than 15 years
4. It verifies if the participant has basic computer knowledge and meets the age requirement
5. If the participant meets the requirements, the program allows them to join the course. If not,
the program shows a message saying the reqirements are not met.

> Status: The program is recording the information correctly without any errors.
