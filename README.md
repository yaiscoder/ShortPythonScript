# Participant registration

## Description
It is a program to register applicants for a computer course.
It is a program that collects basic information.

## How it works
1. The program asks for the number of participants.
2. For each participant, the program asks for their name, age, and whether they have basic computer skills.
3. The program checks if the participant is older than 15 years.
4. It verifies if the participant has basic computer skills and is over 15 years old.
5. If the participant meets all the requirements, registration is successful. if not, the program shows a message that they do not meet them..
